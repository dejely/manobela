## Reporting a Vulnerability

If you discover a security vulnerability, please report it to us privately.

### How to Report
- Email: **popcornprophets@outlook.com**
- Subject line: **Security Vulnerability Report**
- Include:
  - A clear description of the issue
  - Steps to reproduce
  - Expected vs actual behavior
  - Any proof-of-concept or sample code
  - Affected version(s)

### Response Time
- We will acknowledge receipt within **3 business days**.
- We will provide an initial assessment and expected timeline within **7 business days**.

### What Happens Next
- If the report is confirmed, we will:
  - Work on a fix
  - Release a patch or update
  - Provide guidance for mitigation if immediate patching isnâ€™t possible
- If the report is not reproducible or is not a security issue, we will:
  - Explain why it was declined
  - Provide any relevant feedback or next steps

### Disclosure Policy
- Please keep the vulnerability confidential until we publish a fix.
- We will coordinate public disclosure once a patch is released.
